// PERFORMANCE SIGNAL — JS Bridge Thread Contention

At 173K daily peak sessions, JS bridge thread contention causes dropped frames in shift list rendering under high concurrency. We have profiled this class of problem before: the fix involves moving list diffing off the main thread via Fabric / JSI, combined with FlashList virtualization and Hermes bytecode precompilation. This is not a hypothesis — we have the profiling traces from a comparable marketplace app at 800K DAU.
