Architecture
Offline write queue → Idempotency keys → Flush on reconnect
Local SQLite/Core Data event store. Each clock event assigned a UUID. On reconnect, queue drains in order; server deduplicates by UUID. Android Doze-aware background sync via WorkManager / BGTaskScheduler.
Sprint scope: 3 weeks / 1 mobile engineer

Architecture
Webhook acknowledgment → Background polling fallback → Explicit sync state
On form submit, hold "Awaiting sync" state. Poll ServiceTrade record status at 10s/30s/60s intervals until confirmed complete. If polling fails after 3 attempts, surface actionable error — not silent success. Idempotent resubmit safe.
Sprint scope: 2 weeks / 1 engineer

Architecture
Proactive token refresh → Background heartbeat → Context isolation per job
Refresh token 5 minutes before expiry using background task (iOS BGAppRefreshTask / Android WorkManager). Each job context maintains isolated customer scope — uploads, notes, and attachments bound to active job ID, preventing cross-customer attribution. Session heartbeat confirms scope integrity every 30 minutes on long shifts.
Sprint scope: 2 weeks / 1 engineer
